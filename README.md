Vaibhav Badoni — Cybersecurity Portfolio
A terminal-style interactive portfolio built with vanilla HTML, CSS, and JavaScript — designed to reflect a cybersecurity mindset through its aesthetic and feel.
Features

Fully interactive terminal interface with real command support
Animated background with matrix rain, drifting security commands, and a live hex node network
Tab autocomplete and command history navigation (↑↓ arrows)
Responsive design — works on both desktop and mobile
Zero dependencies — pure HTML/CSS/JS, no frameworks
