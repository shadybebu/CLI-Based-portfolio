# Vaibhav Badoni — Cybersecurity Portfolio

A terminal-style interactive portfolio built with vanilla HTML, CSS, and JavaScript — designed to reflect a cybersecurity mindset through its aesthetic and feel.

---

## Features

- Fully interactive terminal interface with real command support
- Animated background with matrix rain, drifting security commands, and a live hex node network
- Tab autocomplete and command history navigation (↑↓ arrows)
- Responsive design — works on both desktop and mobile
- Zero dependencies — pure HTML/CSS/JS, no frameworks

---

## Commands

| Command   | Description                          |
|-----------|--------------------------------------|
| `ls`      | List all available sections          |
| `about`   | Summary and focus areas              |
| `resume`  | Education & certifications           |
| `projects`| Projects with GitHub links           |
| `skills`  | Domain expertise                     |
| `tools`   | Tech stack & security tools          |
| `contact` | Links to email, LinkedIn, TryHackMe  |
| `clear`   | Clear the terminal                   |

---

## Tech Stack

- **HTML5 Canvas API** — animated background (matrix rain, hex node network, drifting commands)
- **CSS3** — scanlines, vignette, glow effects, blinking cursor, animations
- **Vanilla JavaScript** — terminal engine, command parser, history, autocomplete
- **Share Tech Mono** — Google Fonts
